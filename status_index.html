Status Update Tools

This module provides access to the development version of a collection of tools updating status and producing selected data files for exchange with depositors.

Latest version updated: Aug 31, 2015

  • Release status set in workflow tracking database and in the content database
  • Communication module embedded
  • Options provided to create PDB, PDBx with public data subset, and assembly files
  • Reports provided for model file content and model file dictionary checks
  • Add tab for EMDB status
