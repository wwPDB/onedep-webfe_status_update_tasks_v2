On-line Help for Status Update Tools

Deposition data set code

This option allows you to set the depositon data set code in standalone mode. If you see this form then you must set the data set code before using other options.

Update release status.

This option allows you to update the release status and approval type in the model file, the tracking database, and status content database.

Create data files

The option allows you to create data files of the following types: PDB format, PDBx format with public data subset, and assembly model files. Assembly files are generated using assignments in the data categories pdbx_struct_assembly/pdbx_struct_assembly_gen.

View reports

The option allows you to create reports displaying the content of the data model file as well as dicitionary check reports for this file.

Merge coordinates

The option allows you upload and merge replacement coordinates within the current model file. The model file with merged coordinates will be written to the archive as a the next version of this file.
